クイズアプリしITFの 勉強できるように 試験 にも使える。
![Start](./assets/127.0.0.1_5500_Index.html%20(1).png)

![Start](./assets/127.0.0.1_5500_Index.html%20(2).png)

答えが　正しいが　間違いのも　見せます。
画面の上に　正しい　答え　教えます。

![Start](./assets/127.0.0.1_5500_Index.html%2520(4).png)
